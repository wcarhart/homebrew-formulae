# Homebrew Tools
This tap manages [wcarhart's](https://github.com/wcarhart) tools that are installable via Homebrew.

## Available Formulae
### Koi
Repository: https://github.com/wcarhart/koi
```
brew tap wcarhart/tools
brew install koi
```
